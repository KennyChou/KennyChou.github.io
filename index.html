TamperMonkey for eBird 腳本

發表於 2022-05-04 更新於 2022-10-26

前題提要

因 eBird 的賞鳥紀行的功能開放，提供可透過網址抓取全部的 Checklist 記錄編號，有人提出是否可以寫程式一次抓取全部資料內，像 MYEBIRDDATA 的完整格式。

https://ebird.org/tripreport-internal/v1/locations/[TRIP ID]

因上一版 eBird for chrome extension 沒上架，造成安裝及更新上的不便，就想到好像有 Chrome extension 的工具是寫腳本，可以更改特定網頁的長相或者加功能，就進一步搜尋，找到 Tampermoneky 這個工具，然後順便 google 有沒別人寫好的 vuejs 版的範本，可以加以利用。

Tampermoneky for eBird 腳本

先安裝 Chrome 外掛 TamperMoneky
安裝 eBird 賞鳥紀行腳本
安裝 eBird 記錄清單腳本
安裝 eBird 軌跡腳本

安裝成功會要求輸入 eBird API 金鑰 跟先下載鳥名錄

設定完後就會有下載功能按鈕

已知問題: 清單內若有敏感性物種會無法取得。

eBird記綠下載工具

發表於 2021-12-14 更新於 2022-10-26

ebird 記錄小工具(chrome extension)

因為自己無聊寫程式練功,沒上架所以要自行安裝

安裝方式

  1. 下載ebird-export-v1.0.5.zip,存儲在電腦,並解開壓縮檔

  2. 至 chrome\設定\更多工具\擴充功能
    開啟開發人員模式,
    載入未封裝項目,選取剛開壓縮檔目錄, 就會安裝成功

  3. 開網頁至 ebird checklist 的網頁, 按下剛安裝成功外掛的按鈕, 就會看到功能

  4. 目前已加入功能,可複製至剪貼簿, 跟匯出成 excel 或 csv 格式檔案, excel 可設定同頁或分頁

兩棲志工蛙類調記錄器

發表於 2021-12-14 更新於 2022-11-02

前言

兩棲志工野外記錄時, 一定很常遇到天氣不佳, 又濕或又雨的天氣, 常常要撐一把傘口裡咬著手電筒, 還要拿著記錄紙版, 小心不讓它們濕掉。
重點是回家後的紙本資料上傳又是一個另人開心不起來的操作, 還要猜那天晚上到底記錄是什麼蛙, 自己寫的字還考到自己。若筆數過多, 又要慢慢點滑鼠到天明。

相信應該不少團隊有不同私藏記錄方法及上傳工作。

如果你已對紙本記錄熟悉, 後台回傳也熟悉, 相信這工具應該會輕鬆上手, 解決藏在抽屜不想面對的記錄。
